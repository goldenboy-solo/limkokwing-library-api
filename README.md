# Limkokwing Library API System

## Project Description
This project is a simple digital library management API developed using FastAPI for Limkokwing University Sierra Leone.

## Features
- View all books
- View all users
- Borrow books
- Return books
- View borrowed books

## Technologies Used
- Python
- FastAPI
- Uvicorn
- Async Programming

## How to Run
Install dependencies:
```bash
pip install fastapi uvicorn
```
Run the server:

```bash
python -m uvicorn library:app --reload
```
Open in browser:

```text
http://127.0.0.1:8000/docs
```
